# Lambda-Express-Typescript-Babel Starter

### Requirements

- _Node.js v22_ or _Node Version Manager_ (`nvm`).

### Setup

```bash
nvm use
npm install
```

### Test

```bash
nvm use
npm test
```

### Lint

```bash
nvm use
npm run biome
npm run prettier
```

### Run locally

```bash
nvm use
npm start
```
